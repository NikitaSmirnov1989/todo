# Список дел
Здесь будет создаваться простой список дел - для своего личного портфолия на позицию front-end разработчик.

## Функциональность
Список дел будет включить следующую фнукциональность:
* Добавление одной задачи
* Удаление одной задачи
* Удаление всех задач из списка
* Редактирование срока выполнения задачи

## Запуск приложения
Приложение будет запускать при помощи следующий команд
* Скачать удаленный репозиторий к себе на компьютер при помощи команды ниже: 
```
git clone https://github.com/NikitaSmirnov1989/todo
```
* Установить все необходимые зависимости к себе на компьютер
```
npm install
```
* Далее необходимо запустить проект при помози команды 
```
npm start
```
* Далее проект необходимо открыть в адреснйо строке браузера по адресу
```
http://localhost:3000
```
